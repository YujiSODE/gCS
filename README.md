## 1. Synopsis
**Tcl**  
`::gCS::_Run filePath ?encoding?;`  
**Shell**  
`tclsh gCS.tcl filePath ?encoding?;`  

**Parameters**  
- `$filePath`: file path of a file to load
- `$encoding`: encoding name

## 2. Script
It requires Tcl/Tk 8.6+.
